# Postalytics

Postalytics is a direct mail automation platform with a REST API for creating and sending personalized postcards and letters with real-time mail tracking and analytics. The API enables programmatic campaign management for triggered drip campaigns and batch smart-send mailings, with supported SDKs in Python, Node.js, Rust, Java, Go, and C#.
